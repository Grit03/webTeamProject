# 🧱 메이플스토리 Brick Out

## 🖥️ 프로젝트 소개
2022-1학기에 참여한 웹프로그래밍 수업 과제의 일부로 진행한 프로젝트입니다. <br>
4명의 팀원과 협업하여 제작하였으며, 메이플스토리를 테마로 벽돌 깨기 게임을 구현하였습니다.

## 🕰️ 개발 기간
2022.05.24 ~ 2022.06.03
